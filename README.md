**CAP** (Cards, Attendance & Permissions) is a professional **MERN stack** system built to support school operations by simplifying essential tasks such as:

- Generating **student ID cards**
- Managing **permission requests** (leave forms, approvals, tracking)

The system focuses on speed, security, and user-friendly design while supporting future scalability for additional school functionalities.
